# 🛣️ 高速云眼抛洒风险监测平台 (Tarpaulin Detection System)

基于 Flask 与现代前端技术构建的**智慧交通大屏数据可视化平台**。本项目主要用于展示高速公路货车篷布遮盖状态（平板式/栏板式）、实时交通流量、高风险路段监控以及动态抛洒风险预警事件。

系统采用了现代化的**暗黑科技玻璃态 (Glassmorphism)** UI 设计，支持多分辨率屏幕的完美响应式自适应。

---

## ✨ 核心功能与亮点

- 📊 **多维数据统计**：涵盖总流量、各风险级别车辆数、车型占比（ECharts 饼图）以及多日风险趋势（ECharts 折线图）。
- 🗺️ **智慧全景地图**：集成**百度地图 GL 版**，深度定制专属 `midnight`（深海暗黑）科技风地图样式，支持动态点位聚合与风险轨迹线渲染。
- 🚨 **瀑布流预警中心**：右侧风险事件面板采用平滑动画进行“瀑布流”式实时更新，自动高亮不同风险级别的车辆预警截图及定位信息。
- 📹 **实时路况监控**：内嵌高速路段实时视频流/监控录像回放面板。
- 🤖 **AI 智能交互**：后端集成了**百度千帆大模型 (ERNIE-3.5)** 与前端 Dify Chatbot，提供大语言模型智能问答分析支持。
- 🎨 **极致 UI 体验**：全站使用 Flexbox 弹性响应式布局，毛玻璃面板、渐变光效字体、荧光指示灯及随浏览器无缝缩放的图表。

---

## 🛠️ 技术栈

### 后端 (Backend)
- **Python 3.x**
- **Flask**: 提供轻量级 Web 服务与路由跳转。
- **Qianfan (百度千帆)**: 接入文心一言大模型，处理 AI 会话逻辑。

### 前端 (Frontend)
- **HTML5 / CSS3 / JavaScript (ES6)**
- **Bootstrap 5**: 辅助基础样式与组件。
- **jQuery 3.6**: 处理简易的 DOM 动画（如预警列表平滑推送）与 AJAX 轮询。
- **ECharts**: 构建高性能的数据可视化图表。
- **Baidu Map API GL**: 提供高性能的 WebGL 地图与路网渲染。

## 🚀 快速启动

### 1. 环境准备
确保你的电脑上已经安装了 Python 3.8 或以上版本。

克隆或下载本项目到本地，在项目根目录打开终端，安装所需的依赖：
```bash
pip install Flask qianfan
```

### 2. 配置 API 密钥 (可选)
项目中使用了百度地图和百度千帆的 API，如果在生产环境中使用，请替换为你自己的密钥：
- **大模型密钥**：在 `app.py` 中替换 `QIANFAN_ACCESS_KEY` 和 `QIANFAN_SECRET_KEY`。
- **百度地图 AK**：在 `templates/main.html` 及 `main.js` 中替换 `api.map.baidu.com` 链接中的 `ak=你的密钥`。

### 3. 运行服务
在终端执行以下命令启动 Flask 服务器：
```bash
python app.py
```

### 4. 访问大屏
打开浏览器，访问以下地址即可预览：
[http://127.0.0.1:5000/](http://127.0.0.1:5000/)

## 💡 布局与适配说明
本项目的前端架构经过深度优化：
1. **彻底告别白边与溢出**：采用 `height: 100vh` 与 `min-height: 0` 魔法打破了 Flex 盒子的默认高度撑开机制。
2. **文字动态缩放**：核心数据指标使用了 CSS `clamp()` 函数与 `vh` 视口单位，在 13 寸轻薄本到大尺寸无缝拼接屏上均能保持完美比例。
3. **图表自适应**：在 `main.js` 底部已绑定了 `window.addEventListener('resize')`，调整浏览器窗口大小时图表会自动重绘防畸形。
